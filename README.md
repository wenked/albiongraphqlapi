Graphql wrapper to Albion online official API , this wrapper helps to find battles and zerg compositions of each guild in battles . 


Ex:
`
BattleList
{
  battleList(guildName:"R A I D",offSet:20){
    alliances{
      name
      kills
      killFame
      id
      deaths
    }
    endTime
    startTime
    totalFame
    totalKills
    totalPlayers
    id
    winnerAllys
    losersAllys
    winnerGuilds
    losersGuilds
    
     guilds{
      name

      allianceId
      deaths
      id
      killFame
      kills
      name
      
    }
  }
} `
